# Spacing & Responsive Guidelines

## Container Spacing

- Mobile (< 640px): `px-4` (16px)
- Tablet (640-1024px): `px-6` (24px)
- Desktop (> 1024px): `px-8` (32px)

## Section Spacing

- Between sections: `py-12 md:py-16 lg:py-20`
- Inside sections: `space-y-8 md:space-y-10 lg:space-y-12`

## Component Spacing

- Card padding: `p-4 sm:p-6 lg:p-8`
- Button height: `h-9 sm:h-10 lg:h-11`
- Input height: `h-10 sm:h-11`
- Gap between items: `gap-4 sm:gap-6 lg:gap-8`

## Typography

- Headings:
  - H1: `text-2xl sm:text-3xl lg:text-4xl xl:text-5xl`
  - H2: `text-xl sm:text-2xl lg:text-3xl`
  - H3: `text-lg sm:text-xl lg:text-2xl`
- Body: `text-sm sm:text-base`
- Small: `text-xs sm:text-sm`

## Breakpoints (Tailwind)

- sm: 640px
- md: 768px
- lg: 1024px
- xl: 1280px
- 2xl: 1536px

## Touch Targets (Mobile)

- Minimum: 44x44px (h-11 w-11)
- Recommended: 48x48px (h-12 w-12)
